Synapse.js

Synapse.js is a javascript/node.js neural networks library, its generalized algorythm is architecture-free, so you can build and train basically any type of first order or even second order neural network architectures.

This library includes a few built-in architectures like multilayer perceptrons, multilayer long-short term memory networks (LSTM) or liquid state machines, and a trainer capable of training any given network, and includes built-in training tasks/tests like solving an XOR,passing a Distracted Sequence Recall task or an Embeded Reber Grammar test.

Demos:

  • Solve an XOR.
  • Distracted sequence recall task.
  • Generate text.
